SessionHandler

SessionHandler is a class that allows you to store and retrieve data for a user across multiple pages. Uses a MongoDB to store Sessions.

Requires

mongodb, Logger

Constructor

new SessionHandler ( mongoDBInstance )

Methods

  • createSession ( username ) 
    @param username: string - username of the user
@return void: creates a new session for specified User
  • checkSession ( req, res, next ) 
    @param req:    express.js Request
@param res:    express.js Response
@param next:   express.js Next
@return void: calls next() if session is valid
  • destroySession ( username ) 
    @param username: string - username of the user
@return void: destroys the session of the specified user
  • updateSession ( username ) 
    @param username: string - username of the user
@return void: updates the session of the specified user
